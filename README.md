PROJECT OVERVIEW

  This project is a redesigned version of the Archive of Our Own (AO3) interface, focused on improving readability, navigation, and user immersion. The prototype was developed using JAVAFX and demonstrates key UI/UX improvements such as adaptive themes, simplified navigation, and a cleaner layout.

REQUIREMENTS
- Java JDK 21
- Maven
- IntelliJ IDEA (recommended)

HOW TO RUN THE PROJECT

1. Open the project in IntelliJ IDEA
2. Open the Maven panel (right side)
3. Navigate to Plugins -> javafx -> javafx:run

FEATURES DEMONSTRATED
- Light/Dark reading mode
- Genre-based immersive UI concept
- Collapsible side bar navigation
- Simplified browsing and search layout
